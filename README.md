## ğŸ“Š Main Categories

### Reconstruction-Based Methods
- **Autoencoders (AE)** - Learn to reconstruct normal data; anomalies have high reconstruction error
- **Variational Autoencoders (VAE)** - Probabilistic approach with better generalization
- **Generative Adversarial Networks (GAN)** - Generator vs. Discriminator for complex distributions
- **Transformers** - Self-attention mechanisms for sequence anomalies

### Prediction-Based Methods
- **RNN/LSTM/GRU** - Predict next values in sequences; prediction errors indicate anomalies
- **Temporal Convolutional Networks** - 1D convolutions for time series
- **Attention Mechanisms** - Focus on relevant temporal patterns
